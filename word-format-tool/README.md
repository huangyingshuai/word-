专为大学生竞赛/毕业论文/办公人员打造的Word文档一键标准化排版工具，解决手动调格式费时费力的痛点。

## ✨ 核心功能
- 零误判智能标题识别，自动分级分类
- 专业模板一键套用，覆盖毕业论文、竞赛报告、党政公文等场景
- 正文数字/英文单独格式设置，符合规范要求
- 图片100%完整保护，杜绝变形、丢失
- 多文档批量处理，大幅提升效率
